¿Qué es?

FacturaScripts2020

FacturaScripts es un software administrativo y contable, con características de un software de Planificación de recursos empresariales (ERP) Open source para la pequeña, mediana empresa, autónomos o independientes, es un software en la nube de tipo modular para llevar a cabo la facturación online de su negocio.

Además de ser software libre, cuenta con un potente sistema de plugins que le permiten personalizar su funcionamiento, es decir, es FacturaScripts el que se adapta a su negocio y no al contrario.
